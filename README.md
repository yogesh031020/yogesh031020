### 📡 System Overview & Core Mission
I am an **Aeronautical & Robotics Systems Engineer** passionate about building autonomous flight stacks, robust control algorithms, and failure-resilient robotic ecosystems. My work bridges the gap between aerospace physics and high-performance software systems. Formerly at **Novatech Robo Pvt Ltd**, I engineered and deployed commercial-grade autonomous UAV flight control stacks, implemented real-time sensor calibration routines, and conducted black-box telemetry logging and analysis.

I specialize in **bare-metal embedded firmware (C++)**, real-time task scheduling, **ROS 2** distributed stacks, and advanced control systems (LQR, PID, EKF sensor fusion) designed to operate reliably under real-world physical constraints.
