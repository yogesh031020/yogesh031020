# Yogesh E S — UAV & Embedded Systems Engineer

Aeronautical engineer with 2 years building autonomous drone systems — from MAVLink firmware on ESP32/APM 2.8 to deep learning-based navigation in AirSim. I work across the full stack: embedded C++, flight controller integration, sensor fusion, and Python-based autonomy.

---

## What I build

- **Autonomous UAVs** — GPS-denied indoor navigation using IR beacons, A* pathfinding, and optical flow
- **Embedded firmware** — Real-time C++ on ESP32 talking MAVLink to APM 2.8 flight controllers
- **AI-powered flight** — YOLOv8 detection, CNN-based obstacle avoidance, edge ML inference on microcontrollers
- **Ground control systems** — Web HUDs, telemetry dashboards, mission planning tools via DroneKit/MAVProxy
